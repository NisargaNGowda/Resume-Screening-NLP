# Resume Screening Using NLP

## Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify resumes into professional categories.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- TF-IDF Vectorization
- Multinomial Naive Bayes

## Dataset
- Real-world Resume Dataset from Kaggle
- 2484 resumes
- 24 job categories

## Features
- Reads resume text from a file
- Converts text into TF-IDF features
- Predicts job category
- Uses Machine Learning classification

## Model
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Accuracy
56.94%

## Categories
HR, Finance, Banking, Engineering, IT, Healthcare, Teacher, Business Development and more.
